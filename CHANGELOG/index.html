Changelog

Warning

This file documents notable changes to this project done before December 2024. For changes after that date, please refer to the release notes of each release at https://github.com/icub-tech-iit/ergocub-software/releases.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

[0.7.6] - 2024-12-06

  • Add nice looking meshes for both SN000 and SN0001/2 ergocub models (icub-tech-iit/ergocub-software!266)
  • Fixed waist_imu_0 position (icub-tech-iit/ergocub-software!267)

[0.7.5] - 2024-09-03

  • Fix alljoints inertials to work also on gz-sim (icub-tech-iit/ergocub-software!250)
  • ergoCub1_0: fix mass and inertia of the head (icub-tech-iit/ergocub-software!249)
  • Add angry emotion and not fullscreem mode to ergoCubEmotions (icub-tech-iit/ergocub-software!258)

[0.7.4] - 2024-05-27

  • Refactor YAMLs for using the includes creo2urdf parameter(icub-tech-iit/ergocub-software!240)
  • Add ergoCubSN002/V1_2 (icub-tech-iit/ergocub-software!242)

[0.7.3] - 2024-04-10

  • Added xml files for running the publish of physical joints quantities (icub-tech-iit/ergocub-software!227)
  • ergoCubSN001/ergoCubGazeboV1_1*: fix orientation of inertias (icub-tech-iit/ergocub-software!226)
  • Added gz-sim support (icub-tech-iit/ergocub-software!230)

[0.7.2] - 2024-03-11

[0.7.1] - 2024-02-29

  • ergoCubV1_1*: fix feet imus placement (icub-tech-iit/ergocub-software!222)

[0.7.0] - 2024-02-28

  • Expose imu of the ft sensors in the feet(icub-tech-iit/ergocub-software!217)
  • ergoCubEmotions now consume less cpu (icub-tech-iit/ergocub-software#213)
  • ergoCubV1: fixed the names r/l_upperarm with r/l_upper_arm(icub-tech-iit/ergocub-software#197)
  • Fixed HF when resetting the world when using DART as PE(icub-tech-iit/ergocub-software#190)
  • Added couplingXCubHandMk5 device

[0.6.0] - 2023-11-15

urdf

  • Fixed location of l/r_sole frames(icub-tech-iit/ergocub-software#186)
  • Fixed alignment of depth to the RGB frame(icub-tech-iit/ergocub-software#183)

[0.5.0] - 2023-10-26

urdf

  • Fixed wobbling of the models after moving feet link frames

[0.4.0] - 2023-09-04

urdf

  • Added ergoCubGazeboV1_1/_minContacs ergoCubSN001
  • Repository refactored for using creo2urdf

[0.3.4] - 2023-08-28

urdf

  • ergoCubGazeboV1: reenabled torso_pitch (icub-tech-iit/ergocub-software#152)
  • ergoCubGazeboV1: "center" feet csys w/ ft csys (icub-tech-iit/ergocub-software#146)

[0.3.3] - 2023-07-07
